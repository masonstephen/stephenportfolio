# CV & Favicon assets

Place these files in this folder:

| File | Purpose |
|------|---------|
| `Stephen_Mason_CV.pdf` | Your resume (previewed at `/resume`) |
| `favicon.png` | Profile photo used as the site favicon (square, at least 192×192px recommended) |

If your photo is a `.jpg`, either rename it to `favicon.png` or update `faviconUrl` in `src/data/portfolio.js`.
