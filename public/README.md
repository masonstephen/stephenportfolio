# Public assets

## Open Graph image (social & Google previews)

Add a share image at `public/og-image.png` (recommended 1200×630px).
If missing, add your profile photo as `favicon.png` and duplicate/rename for social sharing.

## Favicon (profile photo)

Place your photo in this folder (`public/`) using one of these names:

- `favicon.png` (recommended)
- `favicon.jpg`
- `profile.png`
- `profile.jpg`

The site auto-detects the first match. Square images work best (192×192px or larger).
